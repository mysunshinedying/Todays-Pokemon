# 오늘의 포켓몬은 뭘까요? Today's Pokemon

오늘의 포켓몬을 뽑고, 포획하고, 럭키 아이템도 가져보세요!


## 주요 기능
- 오늘의 포켓몬을 뽑기
- 포켓몬을 포획해 박스에 넣기
- 포켓몬 정보를 확인하기
- 포획한 포켓몬의 럭키 아이템을 보기
- 진행 사항을 저장

## 기술 스택
- HTML5, CSS3, JavaScript
- React
- pokeAPI 사용

## 현재 업데이트
- 출석 체크 및 localStorage에 뽑은 포켓몬을 저장합니다.
- API를 불러와 포켓몬을 뽑고, 포획 후 럭키 아이템을 뽑습니다.
- 프로젝트가 생성되었습니다.
